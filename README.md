## Описание 📜

Данный проект представляет собой пример моей работы, выполненной в компании **Metamodern**. Приложение разработано с использованием **UIKit** и представляет собой календарь с возможностью импорта событий из системного календаря iOS, создания собственных событий, а также настройки виджетов для рабочего стола с помощью **WidgetKit**.

В проекте реализованы работа с календарем через **EventKit**, локальное хранение данных, поддержка уведомлений и синхронизация данных между приложением и виджетами с использованием **App Groups**.
